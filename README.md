# 家計簿アプリ
PHP自作

## 概要
家計簿アプリを作成しました。

管理者ユーザーと一般ユーザーに分け、 それぞれでログインできるようにしています。

所持金・収入・支出を確認できる機能があります。

## 使い方
### 管理者ユーザー

このアプリのユーザーを一覧表示で確認できるようにしています。
